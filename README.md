## 📌 Overview

A modern and fully functional **Task Management Web Application** built using the **MERN Stack**  
(MongoDB, Express.js, React.js, Node.js).

This application helps users manage daily tasks efficiently with secure authentication, drag-and-drop task handling, and a responsive user interface.

---

# ✨ Features

✅ JWT Authentication & Authorization  
✅ Google OAuth Login  
✅ Create, Update, Delete & Search Tasks  
✅ Drag & Drop Task Management  
✅ Optimistic UI Updates using React Query  
✅ Fully Responsive Design  
✅ Protected Routes  
✅ Secure HttpOnly Cookies
